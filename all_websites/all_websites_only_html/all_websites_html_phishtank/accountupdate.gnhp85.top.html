    注意事項
    • MyJCB ID・パスワードは、半角英数字・記号6〜20桁でご入力ください。
    • アルファベットは大文字･小文字をご確認のうえ、正しく入力してください。
    • 入力エラーが当社規定の回数を超えると、セキュリティー保持のためそのIDは利用できなくなり、あらためてMyJCBの新規登録が必要となります。

    MyJCB IDをお持ちでない方、ID・パスワードをお忘れの方やログインできない方は、下の「新規登録／ID確認」ボタンよりお手続きください（登録は無料です）。

MyJCBのご利用にあたって

  • サイトのご利用について

    • MyJCBではインターネット通信で利用する暗号化方式「TLS」を使用しております。なお、アクセス環境によってTLSが確立しない場合は、サービスを利用できません。
    • MyJCBでは「Cookie」を使用しておりますので、ブラウザの設定でCookieを有効にしていただく必要があります。
    • 動作環境については「サイトのご利用について」をご参照ください。
    サイトのご利用について

  • 「秘密の合い言葉」によるご本人様確認について

    MyJCBを安心してご利用いただくため、ログイン時に普段と異なるご利用環境と判断した場合、「秘密の合い言葉」によるご本人様確認を実施しています。

    「秘密の合い言葉」が未登録の場合、ログイン後に登録画面が表示されます。

  • MyJCBのID・パスワードの管理について

    MyJCBのID・パスワードを第三者に譲渡または使用させる行為は禁止しています。

    当社が定める特定加盟店以外に、他社のサービスでMyJCBのID・パスワードの入力を求めるものがありますが、不正使用され、思わぬ被害にあってしまう危険性があります。ID・パスワードは厳重に管理してください。
